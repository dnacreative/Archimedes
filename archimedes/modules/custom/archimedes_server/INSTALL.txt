-- REQUIREMENTS --

* The python-webkit2png library.

* The following pacakges:

    libqt4-webkit
    pythonqt4
    xvfb (for servers not running x)


-- INSTALLATION --

* Download and install the required packages. On Debian/Ubuntu you can use the following line:

    apt-get install libqt4-webkit python-qt4 xvfb

* Copy the archimedes server module directory to your sites/all/modules directory, so it
  is located in sites/all/modules/archimedes_server.

* Download the python-webkit2png release from:

  https://github.com/AdamN/python-webkit2png/tarball/master

* Put the downloaded archive into the module directory:

    /sites/all/modules/archimedes_server/AdamN-python-webkit2png-4e625af.tar.gz

* Extract the archive to python-webkit2png.  This will create the following sub-directory:

    /sites/all/modules/archimedes_server/python-webkit2png/
